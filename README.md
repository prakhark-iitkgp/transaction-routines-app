# transaction-routines-app
This application Contains APIs to perform transaction routines
- POST /accounts -> To create an account
- GET  /accounts/{accountId} -> To obtain the details of an account using the accountId
- POST /transaction -> create a transaction for a given accountId, amount and operationType;

# Steps to run the application

- Open a bash terminal and move to the repository root
- With docker running, execute the run script by executing command ./run.sh
